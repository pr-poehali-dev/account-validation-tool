# Сборка десктопного приложения для Windows (.exe)

Программа упаковывается в Windows-приложение через Electron.
Всё нужное уже добавлено в проект — осталось выполнить несколько шагов на своём компьютере.

## Что уже готово в проекте
- `electron/main.cjs` — главный процесс Electron (окно + локальный сервер для собранных файлов)
- `electron/preload.cjs` — мост между интерфейсом и проверкой
- `electron/checker.cjs` — АВТОНОМНАЯ проверка аккаунтов прямо на компьютере (без облака)
- `electron-builder.json` — настройки сборщика установщика (.exe)

В десктоп-версии проверка идёт локально: программа сама ходит на сайт,
работает с прокси и потоками без облачной функции. В шапке индикатор **LOCAL**.
В браузере (на сайте) остаётся облачный режим — индикатор **CLOUD**.

## Шаг 1. Скачай код проекта
В интерфейсе платформы: **Скачать → Подключить GitHub** (или **Скачать код**).
Распакуй проект на компьютере с установленным Node.js (версия 18+).

## Шаг 2. Установи зависимости
Открой папку проекта в терминале и выполни:

```
npm install
npm install --save-dev electron electron-builder concurrently wait-on
npm install https-proxy-agent socks-proxy-agent
```

Последние два пакета нужны для работы прокси в автономном режиме.

## Шаг 3. Добавь скрипты в package.json
Файл `package.json` на платформе защищён от изменений, поэтому добавь вручную
после скачивания. В раздел `"scripts"` добавь строки:

```json
"electron:dev": "concurrently \"vite\" \"wait-on http://localhost:5173 && electron .\"",
"electron:build": "vite build && electron-builder --config electron-builder.json"
```

А в корень файла (рядом с "version") добавь:

```json
"main": "electron/main.cjs"
```

## Шаг 4. Запуск в режиме разработки (проверить, что всё работает)
```
npm run electron:dev
```
Откроется окно приложения с горячей перезагрузкой.

## Шаг 5. Собрать установщик .exe
```
npm run electron:build
```
Готовый установщик появится в папке `release/`.
Это будет файл вида `T4Trade Checker Setup x.x.x.exe` — его можно
устанавливать на любой Windows и запускать как обычную программу.

## Иконка приложения (уже подключена)
Путь к иконке `build/icon.ico` уже прописан в `electron-builder.json`.
Осталось положить сам файл — выполни 2 шага:

1. Скачай готовую иконку:
   https://cdn.poehali.dev/projects/53675d38-34c3-42f1-a6b5-bb14c5a2c52e/files/abb9962a-8cbb-427c-b0a0-1c8f133e78c0.jpg

2. Конвертируй её в формат `.ico` (256×256) — например на https://icoconvert.com
   и сохрани как `build/icon.ico` в папке проекта.

После этого собранный `.exe` будет с фирменной иконкой.

## Автономная проверка
Десктоп-версия проверяет аккаунты прямо на компьютере — без облачной функции.
Это работает автоматически: интерфейс определяет, что запущен внутри Electron,
и использует локальный движок `electron/checker.cjs`. Прокси, потоки и ротация
работают так же, как в облаке. В шапке программы будет гореть индикатор **LOCAL**.
